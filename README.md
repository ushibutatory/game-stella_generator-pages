## なにこれ

- ランダムに表示される星をつないで、星座の形を作ります。
- 作成した星座の形をもとに、AI が星座の名前を考えます。
- 3 歳の娘が遊ぶ想定なので、テキストは敢えてひらがなを多用しています。
  - AI が生成するテキストには難しい表現が含まれることがあります。
